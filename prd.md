# PRD: Дашборд СМИ Казахстана

## 1. Обзор

Одностраничный сайт (single HTML file), объединяющий три источника информации о медиасреде Казахстана:
1. Рейтинг СМИ по цитируемости/посещаемости
2. Справочник экспертов по тематическим направлениям с их соцсетями
3. Каталог региональных СМИ по областям

**Статус:** реализовано, файл [`index.html`](index.html)

## 2. Проблема / цель

Нет единого места, где можно посмотреть:
- какие казахстанские СМИ лидируют по цитируемости/посещаемости,
- кто из экспертов авторитетно пишет про конкретную тему (инвестиции, госзакупки и т.д.) и где на него подписаться,
- какие СМИ существуют в конкретном регионе РК и где их сайты.

Цель — собрать всё в одном доступном файле без необходимости разворачивать сервер или базу данных.

## 3. Целевой пользователь

Человек, работающий с казахстанским медиаполем: PR/GR-специалист, журналист, маркетолог, исследователь — ищет быстрый ориентир по СМИ и экспертам, без цели получить готовую аналитическую систему.

## 4. Требования к продукту (реализованные)

### 4.1 Формат
- Один файл `.html`, без сборки/бэкенда, открывается в любом браузере.
- Верстка на чистом HTML/CSS/JS, без внешних зависимостей.
- Адаптивная, с sticky-навигацией по вкладкам.

### 4.2 Вкладка «Рейтинг СМИ»
- Таблица топ-20 СМИ Казахстана по посещаемости — данные получены напрямую с LiveInternet через их публичный TSV-эндпоинт (`liveinternet.ru/rating/kz/media/month.tsv`, категория «Новости и СМИ», гео Казахстан, период 31 день). Интерактивный виджет LiveInternet встроить как iframe не удалось (страница требует ручного выбора фильтров через JS, хэш-параметры в URL не читаются при загрузке) — вместо этого используется снимок реальных данных с указанием даты и прямой ссылкой на источник.
- Блок «Подтверждённые точечные факты 2023–2025» — отдельные проверенные факты о позициях СМИ (Tengrinews.kz, Sports.kz, 24.kz, Петропавловск.news), каждый со ссылкой на источник и датой.
- Блок «источники, которые не дали рейтинга» — прозрачный список проверенных, но безрезультатных источников (Ranking.kz, МИОР РК, Медиалогия, ZERO.kz, JagaJam и др.).

### 4.3 Вкладка «Эксперты по темам»
- 8 категорий: инвестиции/фондовый рынок, госзакупки, макроэкономика, IT, недвижимость, банковский сектор, нефтегаз, политика/госуправление.
- На каждого эксперта: имя, краткое описание (должность/деятельность), ссылки на реальные соцсети (Telegram, Facebook, Instagram, LinkedIn, X), источник, откуда взяты данные.
- Поле поиска, фильтрующее по всем категориям одновременно (по имени, описанию, соцсетям).
- Записи без подтверждённого аккаунта помечены чипом «не подтверждён» — вместо придуманных ссылок.
- Репутационные риски отмечены явно (например, критика в адрес одного из инвестблогеров).

### 4.4 Вкладка «Региональные СМИ»
- ~20 групп по регионам (17 областей + Алматы, Астана, Шымкент).
- В каждой группе: название СМИ, тип (сайт/газета/ТВ), ссылка на официальный сайт.
- Поле поиска по названию региона или СМИ.
- Приоритет отдан сети областных филиалов АО «РТРК Казахстан» и локальным изданиям; общенациональные агрегаторы (Tengrinews, Zakon.kz и т.п.) намеренно не включены как «региональные».

### 4.5 Прозрачность данных (сквозное требование)
- Каждая единица данных (СМИ, эксперт, факт) сопровождается ссылкой на источник.
- Не допускается фабрикация: если факт/аккаунт не подтверждён — указывается это явно, а не заполняется правдоподобной догадкой.
- На сайте указана дата сбора данных как контекст актуальности.

## 5. Нефункциональные требования
- Работает офлайн после загрузки (кроме переходов по внешним ссылкам).
- Не требует npm/build-процесса — статический HTML+CSS+JS.
- Поиск/фильтрация — мгновенные, на стороне клиента (без бэкенда).

## 6. Источники данных (методология сбора)
Данные собраны через веб-поиск (WebSearch/WebFetch) тремя независимыми проходами:
1. Рейтинг СМИ — поиск публичных исследований цитируемости/посещаемости (Brand Analytics, LiveInternet, Ranking.kz и др.).
2. Эксперты — поиск по каждому направлению отдельно, с проверкой реальных профилей в соцсетях.
3. Региональные СМИ — обход по всем регионам РК через справочники, Wikipedia, официальные сайты филиалов РТРК.

## 7. Известные ограничения
- Таблица посещаемости — это статичный снимок на дату сбора (LiveInternet не даёт публичного CORS-доступа для live-запроса из браузера со стороннего домена), при необходимости требует ручного обновления по той же ссылке на TSV-эндпоинт.
- Часть аккаунтов экспертов не подтверждена напрямую (пометка в интерфейсе).
- Число подписчиков/активность каналов могут быть неактуальны на момент просмотра.
- Направления «недвижимость» и «госзакупки» слабо представлены персональными экспертами — больше материала от образовательных платформ/изданий, чем от конкретных лиц.
- Список региональных СМИ не претендует на полноту (не исчерпывающий реестр всех СМИ РК).

## 8. Вне рамок (out of scope)
- Автоматическая интеграция с API LiveInternet/Яндекс.Метрики (нет доступа к приватным счётчикам конкретных изданий).
- Бэкенд, база данных, авторизация.
- Автообновление данных по расписанию.

## 9. Возможные следующие шаги
- Ручная верификация всех записей с пометкой «не подтверждён» перед публичным использованием.
- Периодическая ревизия рейтинга СМИ при появлении новых открытых исследований.
- Расширение списка экспертов и региональных СМИ по мере поступления новых данных.
