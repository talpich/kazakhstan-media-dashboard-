# СМИ Казахстана: рейтинг, эксперты, регионы

Одностраничный сайт-дашборд по казахстанской медиасреде:

- 📊 **Рейтинг СМИ** — цитируемость/посещаемость с указанием источников и честным дисклеймером там, где актуальных данных нет
- 🎓 **Эксперты по темам** — инвестиции, госзакупки, макроэкономика, IT, недвижимость, банки, нефтегаз, политика — с реальными соцсетями и источниками
- 📍 **Региональные СМИ** — список изданий по всем регионам Казахстана с сайтами

## Использование

Открыть [`index.html`](index.html) в браузере — сборка не требуется.

Если репозиторий публичный, можно включить GitHub Pages (Settings → Pages → Deploy from branch → `main` / `/`) — сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

## Документация

Требования и методология сбора данных описаны в [`prd.md`](prd.md).

## Важно

Данные собраны через открытый веб-поиск и не гарантированно полны/актуальны. Записи с пометкой «не подтверждён» и полнота рейтинга СМИ 2024–2026 требуют дополнительной проверки — подробности в `prd.md`, раздел «Известные ограничения».
